Welcome to Markdown for Agents

This is a sample page to test HTML-to-markdown conversion via AWS CloudFront.

How It Works

When an AI agent sends a request with Accept: text/markdown, the CDN automatically converts HTML to markdown on the fly.

Features

  • Automatic HTML to markdown conversion
  • Cache-aware: converted markdown is cached separately from HTML
  • Token count estimation via x-markdown-tokens header
  • Fallback to HTML if conversion fails

Supported Elements

  1. Headings (h1-h6)
  2. Paragraphs and inline formatting
  3. Ordered and unordered lists
  4. Links and images
  5. Code blocks and inline code
  6. Tables

Example Code

curl -H "Accept: text/markdown" https://example.com/page.html

Architecture Overview

The system uses a CloudFront distribution with a viewer-request function that detects the Accept header and routes markdown requests to a Lambda function.
